# ❤️ Heart Stroke Disease Prediction

🚀 **An end-to-end Machine Learning web application** that predicts the risk of heart stroke disease based on user health data.
Built with **Streamlit, Python, and Scikit-learn** with a modern UI design.

---

## 📌 Project Overview

This project uses a **K-Nearest Neighbors (KNN)** model to analyze health parameters and predict whether a person is at **high risk or low risk of heart stroke**.

The application provides:

* ✅ Real-time prediction
* ✅ Clean and interactive UI
* ✅ Data preprocessing (encoding + scaling)
* ✅ Deployment-ready ML pipeline

---

## 🎯 Features

✨ User-friendly web interface (Streamlit)
✨ Tailwind-inspired modern UI
✨ Real-time prediction with probability score
✨ Data preprocessing (One-hot encoding + StandardScaler)
✨ Trained ML model integration using Joblib

---

## 🛠️ Tech Stack

* **Language:** Python 🐍
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Visualization/UI:** Streamlit
* **Model:** KNN Classifier
* **Deployment:** Streamlit Cloud / Local

## ⚙️ How It Works

1️⃣ User enters health details (age, BMI, glucose, etc.)
2️⃣ Data is converted into a structured format
3️⃣ One-hot encoding is applied
4️⃣ Data is scaled using StandardScaler
5️⃣ Model predicts risk using KNN algorithm
